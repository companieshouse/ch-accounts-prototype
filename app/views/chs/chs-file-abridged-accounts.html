File abridged accounts

You can now file abridged accounts here too. While we develop this new service, you should continue to use WebFiling for other types of accounts.

How to use the service

You’ll need to:

  1. enter your company name or number in ‘Search the register’
  2. select ‘File for this company’ on the company overview page
  3. sign in or register to Companies House
  4. enter your company authentication code
  5. select ‘File accounts’

Signing in

If you normally use WebFiling to file your accounts, you won’t be able to use your WebFiling sign-in details here. You’ll have to register for a new password for the new service.

Your company authentication code won’t change.

Get help

If you need any help with the service, please get in touch with us at privatebeta@companieshouse.gov.uk

Your feedback

This is a new service and you can help us improve it by completing our quick survey.
