File abridged accounts to Companies House

There are currently two ways to file abridged company accounts to Companies House:

1. Using the Companies House-HMRC joint filing service:

This will require signing up for a Government Gateway Link opens in a new tab/window account, or using your existing Government Gateway credentials to sign in. This will allow you to file your tax return to HMRC at the same time.

Go to HMRC to use the joint filing accounts service

2. Using third party software

This service is open to everyone, but will mostly benefit those who file frequently - on a daily or weekly basis. The more documents you file, the more appropriate software filing is likely to be.

Go to HMRC to use the joint filing accounts service
