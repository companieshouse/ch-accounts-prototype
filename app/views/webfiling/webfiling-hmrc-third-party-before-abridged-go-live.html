File abridged company accounts to Companies House

We’re working on a replacement service that will enable you to file abridged accounts directly with us. This will be available later this year. Currently there are 2 options to file your abridged accounts online:

1. Using the Companies House-HMRC joint filing service:

You’ll need to use your Government Gateway account or register if you don’t have one. This method lets you file your tax return to HMRC at the same time.

Go to HMRC to use the joint filing accounts service

2. Using third party software

This service is open to everyone, but will mostly benefit those who file frequently - on a daily or weekly basis. The more documents you file, the more appropriate software filing is likely to be.
